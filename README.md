## 🛠️ **보완점 및 후기**

- **고재범**: 처음 도전한 게임 프로그래밍이었지만 다양한 시도를 통해 성장할 수 있었다.  
- **박수진**: 오류로 인해 포기한 기능이 아쉬워 추가 보완할 계획이다.
- **김주범**: 시간이 부족해 시도하지 못한 기능들이 아쉽지만, 이번 경험을 바탕으로 더 나은 결과물을 만들 자신이 있다.
